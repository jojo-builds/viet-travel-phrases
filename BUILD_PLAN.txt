It’s the engine that runs JavaScript outside a browser. Expo needs it. Youinstall it once.What is npm? It’s like an app store for code packages. It comes with Node.js.What is Git? It’s version control — think of it as infinite undo for your entire project. GitHubis the cloud backup.What is VS Code? It’s the text editor where you’ll review code. Think of it as MicrosoftWord for code.What is Expo? It’s a framework that lets you build iPhone apps using JavaScript instead ofSwift. It handles 90% of the hard iPhone stuff for you.What is Expo Go? It’s an app on your physical iPhone that lets you preview your app duringdevelopment without building a full .ipa file. Much faster than building each time.What is EAS Build? It’s Expo’s cloud service that compiles your JavaScript into an actualiPhone app binary. You need this for TestFlight and the App Store.What is TestFlight? It’s Apple’s system for distributing beta apps to testers before they goon the App Store.What is Claude Code? It’s an AI coding agent that runs in your terminal. You describe whatyou want in English, and it reads your files, writes code, and runs commands. This is yourprimary building tool.What is NativeWind? It’s a way to style React Native apps using Tailwind CSS class names(like bg-blue-500, text-xl, rounded-lg). It’s the fastest way to make your app look good.What is AsyncStorage? It’s a simple key-value storage on the user’s phone. Think of it likea tiny local database.
