# Thai Content Draft

This folder is the prep lane for a future Thailand / Thai app surface.

Use it for:

- scenario review against the shared traveler baseline
- Thai-specific phrase planning
- script and pronunciation posture decisions
- prep-only authoring artifacts that make the next translation pass faster
- audio planning after source text is stable

Current status:

- prep-only
- not runtime-wired
- now upgraded from research-only to an authoring-ready first-wave scaffold

Current durable lane assets:

- `research/language-pipeline/thai/THAI-TRAVEL-RESEARCH.md`
  - durable Thailand / Thai synthesis covering script, politeness, scenario emphasis, payment context, and review risks
- `content-draft/thai/phrase-source.csv`
  - reusable source scaffold for baseline plus Thai-specific supplemental rows
- `content-draft/thai/first-wave-priority.csv`
  - ranked first translation wave for the next authoring task
- `content-draft/thai/source-notes.md`
  - authoring stance, pronunciation cautions, and rewrite/review guidance
- `content-draft/thai/research-backlog.md`
  - remaining prep work and graduation blockers

Important lane note:

- Thai should stay prep-only until translation coverage, pronunciation posture, audio posture, and search/localization handling are strong enough for a non-Latin runtime lane.
- The next translation task should start with the top rows in `first-wave-priority.csv`, not with another broad orientation pass.
