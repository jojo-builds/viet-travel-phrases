# Start Session

This file is a compatibility pointer for older startup references.

For broad manual repo sessions, start with:

- `AGENTS.md`
- `docs/DECISIONS.md`
- `docs/PRIORITIES.md`
- `docs/operations/README.md` when the question touches live build, validation, release, or blocker truth

For repo-local queue automation runs, start with:

- `AGENTS.md`
- `.agent/README.md`
- `.agent/QUEUE_START.md`
- `.agent/AUTOMATION.md`

Do not treat this file as a maintained next-step authority.
